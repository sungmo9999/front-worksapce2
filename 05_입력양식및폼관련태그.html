폼 관련 태그

from태그 내의 submit버튼클릭시 해당 from내에 작성된 사용자가 입력한 값들이 서버에 전달된다
[표현법] from action="속성값" method="속성값"

*action 속성: from태그 내부의 입력한 값들을 전달할 서버의 경로
*method 속성: 서버로 전달시 전송방식을 저장하는 속성
>get/post : 두가지 방식이 존재함
1. get: 요청시 사용자가 전달한 값들이 url에 노출되는 방식
2. post : 요청시 사용자가 전달한 값들이url에 노출되지 않는 방식

select 태그와 option태그

- 사용자에게 직접 키보로 값을 입력받는게 아니라 내가 제시한 여러 옵션들 중 하나를 선택 할 수 있게 하는 태그
- 서버로 요청시 현재 선택된(select) 값이 서버로 넘어 간다
- 각 옵션 태그에 velue속성값을 명시한 경우 velue값이 서버로 넘어가고 명시하지 않은 경우 컨텐츠 영역의 값이 서버로 전송된다.
